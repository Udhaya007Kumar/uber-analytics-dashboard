# 🚖 Uber Analytics Dashboard

An interactive data analytics dashboard built using **Streamlit** to analyze Uber trip data and generate business insights.

👉 https://uber-analytics-dashboard-tp78wruu8nenwld5bdbssq.streamlit.app/

---

## 📊 Project Overview

This project focuses on analyzing Uber trip data using SQL and visualizing insights through an interactive Streamlit dashboard.

The dashboard allows users to explore:

- 📈 Total trips and revenue
- 🕐 Peak demand hours
- 💰 Revenue trends over time
- 🌆 City-wise performance
- 💳 Payment method usage
- 🚫 Trip status (completed vs cancelled)

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 📊
- Pandas 🧮
- MySQL 🗄️ (Data Analysis)
- CSV 📁 (Deployment)
